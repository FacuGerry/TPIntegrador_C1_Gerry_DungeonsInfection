# TPIntegrador_C1_Gerry_Titulo

**Itch**: https://facugerry.itch.io/dungeons-infection

**[EN]** Dungeon's Infection is a top-down turn based combat game in which you are Royd, an adventurer who has been traped in a dungeon with his sister, Thane. You'd love to leave and be free, but an infection is spreading across the underworld and you'll need to defeat it in order to escape. Fight against infected explorers and kill all the viruses to set free.

**[ES]** Dungeon's Infection es un juego top-down con un sistema de combate por turnos en el que eres Royd, un aventurero que fue atrapado en una mazmorra con su hermana, Thane. Les encantaría salir y ser libres, pero una infección se está esparciendo por el mundo subterráneo y deben de derrotarla para poder escapar. Lucha contra exploradores infectados y mata a todos los virus para poder conseguir la libertad.

**Controls / Controles**:
- WASD -> Movement / Movimiento
- E -> Interact / Interactuar
- LMB -> Click on buttons to combat enemies / Clickea en botones para combatir a enemigos
- Esc -> Pause / Pausa
